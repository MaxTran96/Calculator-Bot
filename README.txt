README
This is a chatbox which help calculate a simple math problem for the user

---------------------------------------------------------------------------
Requirement
Microsoft Bot Framework Emulator V4 (download the .exe file from the link below)
https://github.com/Microsoft/BotFramework-Emulator/releases
Python 3.6.5+

---------------------------------------------------------------------------
Instruction
Open terminal, and type in 
python main.py
Open the emulator then create a new bot with end point pointed to localhost:9000
For now, its only available for simple math calculations but I'll improve it in 
the future
Enjoy :D
